Type demo

On this page we will use some cool features from type module.

To work with type module we need to set @rhythm ans @base_fs variables.

  @rhythm: 20px;
  @base_fs: 16px;

Next step is to properly set body's font-size.

  body {
    .type_body();      
  }

Now body has 16/20 font.

In header bellow we redefine base font size to 18/20 font. And set hesers sizes from 38 to 18
